# Afeto

Um projeto simples de site web para demonstrar afeto e conexão emocional.

## Descrição

Este é um site básico desenvolvido com HTML e CSS, acompanhado de uma imagem representativa. O objetivo é criar uma página leve e acessível que transmita mensagens positivas.

## Estrutura do Projeto

- `index.html`: Página principal do site
- `styles.css`: Estilos visuais para layout e design
- `afeto.png`: Imagem do projeto

## Como Usar

1. Clone o repositório:
   ```
   git clone git@github-personal:guilhermeborba/afeto.git
   ```

2. Abra o arquivo `index.html` em um navegador web.

## Tecnologias Utilizadas

- HTML5
- CSS3
